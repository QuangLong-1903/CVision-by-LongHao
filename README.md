# CVision – CV Creation & Recruitment Platform

## Description
CVision is a web-based recruitment platform similar to TopCV and VietCV.
The system allows candidates to create professional CVs using predefined templates,
and enables recruiters to post job listings and receive applications directly through the platform.

CVision also integrates an intelligent CV classification module using
Natural Language Processing (NLP) and Machine Learning to automatically
categorize CVs into predefined job fields, supporting recruiters during CV screening.

## Key Features

### For Candidates
- Create and edit CVs online
- Choose from multiple professional CV templates
- Manage personal profiles and uploaded CVs
- Apply directly to job postings on the platform

### For Recruiters
- Recruiter account registration and management
- Create and manage job postings
- Receive and review submitted CVs
- Support candidate screening process

### CV Classification System
- Text preprocessing and feature extraction
- Keyword-based scoring and classification
- Automatic CV categorization by job fields
