# CVision – CV Creation & Recruitment System

## Description
A web-based recruitment platform developed using **Python Flask**.  
The system allows candidates to create professional CVs using templates, apply directly to job postings, and enables recruiters to post jobs and manage applications.

## Key Features
- User authentication and role-based access (Candidate / Recruiter / Admin)
- CV creation with multiple professional templates
- AI-powered text enhancement for CV content
- CV upload and processing
- Job posting and application management
- Application status tracking

## Technologies Used
- Python & Flask
- Flask-SQLAlchemy, Flask-Migrate
- HTML, CSS, JavaScript (Jinja2 Templates)
- SQLite / MySQL
- External AI APIs for CV text enhancement and processing
- Git & GitHub
